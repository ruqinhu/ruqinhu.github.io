# 个人博客转载规范 (Reprint Specification)

为了尊重原作者的知识产权并保持内容的准确性，本博客在转载文章时必须严格遵守以下规范：

## 1. 原文完整性原则
- **禁止修改**：严禁对原文内容进行任何形式的改动，包括但不限于修改文字、删除段落、调整排版（除非是为了适配 Markdown 格式）。
- **完整保留**：必须完整保留原文的所有标题、正文、引用、代码块及相关链接。
- **排版一致**：在 Markdown 转换过程中，应尽可能还原原文的结构层次。

## 2. 来源标注规范
每一篇转载文章的开头必须包含以下元数据信息：
- **转载声明**：明确标注“转载”字样。
- **原作者**：清晰标注文章的原始作者姓名或机构。
- **发布日期**：标注文章在原平台的发布日期。
- **原文链接**：提供可点击的原始文章 URL。

## 3. 技术实施要求
- **自动化解析**：优先使用专门的解析脚本（如 `reprint_tool.py`）进行全量 DOM 解析，避免手工复制导致的格式错误或遗漏。
- **元数据保留**：脚本应自动提取元数据并格式化到 Markdown 头部。

## 4. 样式适配
- 所有转载文章统一采用 `viewer.html` 进行渲染，保持全站视觉风格的一致性。

---
*本规范自 2026 年 4 月 14 日起生效。*
